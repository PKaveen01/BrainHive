# Team Setup Instructions

## For ALL Members:

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/brainhive.git
   cd brainhive

2. Switch to your feature branch:

    Member 1: git checkout feature/user-module

    Member 2: git checkout feature/resources-module

    Member 3: git checkout feature/peerhelp-module

    Member 4: git checkout feature/collaboration-module

3. Install dependencies:

    Backend: Open in IntelliJ, Maven will auto-import

    Frontend: cd frontend && npm install

4. Start your parts:

    Backend: Run BackendApplication.java in IntelliJ

    Frontend: cd frontend && npm start
